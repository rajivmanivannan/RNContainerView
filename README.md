# RNContainerView

RNContainerView is a React Native Android Component to load JS Bundle into an existing Android App.Using this component, We can create an Android and React Native BlendedÂ App.

## Installation

### Development

1. Clone this repo.  
2. Run `yarn install`.  
3. Ensure a device, or emulated Android image is connected (`adb devices`).  
4. Run `react-native run-android`.
